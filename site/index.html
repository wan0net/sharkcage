yeet

Yeet your code tasks at a fleet of Dell 5070s and walk away.

The pitch

You have tasks. You have small computers. You yeet the tasks at the computers.

The computers run AI coding agents -- Claude Code, Crush, Aider -- autonomously. You go get coffee. Or sleep. Or touch grass. Whatever people do when they're not babysitting AI.

When you come back, there are draft PRs waiting for you. Reviewed, tested, ready to merge. Like a very cheap, very quiet development team that never asks about sprint velocity.

If something scary happens -- like the AI wanting to rm -rf / -- yeet pauses and texts you. Because even reckless abandonment has limits.

The stack

What you'd expect

  • Kubernetes
  • 47 microservices
  • A dedicated DevOps team
  • $2,000/month in cloud bills
  • A Slack channel called #incidents

What yeet uses

  • Nomad
  • Some shell scripts
  • Dell thin clients from eBay
  • ~$5/year in electricity per node
  • About 200 lines of CLI code

Orchestration: Nomad. Networking: Tailscale. Provisioning: Ansible. Device routing: udev + flock. Vibes: immaculate.

How it works

You --> yeet run --> Nomad --> Dell 5070 --> AI Agent --> Draft PR --> You (hours later, caffeinated)

That's it. That's the architecture diagram. No, there isn't a second slide.

Features

Agent-agnostic

We don't play favorites. Claude, Crush, Aider -- whoever gets the job done. Bring your own agent if you want. We're not your mom.

Device-aware routing

Need a YubiKey plugged in? yeet knows which Dell has one. It's like Uber but for USB devices.

Cost controls

Set a budget so your AI doesn't spend your rent money on API calls. Capitalism requires guardrails.

Approval gates

If the AI tries something sketchy, yeet pauses and asks you first. Like a responsible adult.

Fleet management

Drain a runner, take it offline, bring it back. Like managing a very small, very cheap data center.

Nomad-powered

We literally just wrapped Nomad's API. Standing on the shoulders of HashiCorp giants.

CLI

# Yeet a task and forget about it
yeet run peer6 "implement the notification system"

# Yeet two tasks at once because you're ambitious
yeet run peer6 "add dark mode" &
yeet run login2 "write all the tests" &

# Check on your fleet of discount workstations
yeet runners

# Oh no, what's it doing
yeet logs peer6-abc123

# STOP STOP STOP
yeet stop peer6-abc123

The Dell 5070

The unsung hero of this whole operation. A thin client so modest it doesn't even have fans.

$50 on eBay
15W your phone charger draws more
$5/yr electricity cost
0 dB silent, fanless
24/7 runs forever

It fits in a desk drawer. It runs Ubuntu. The AI runs in the cloud anyway -- these just manage repos, shuttle files, and look unassuming. More computing power than the job requires, which is exactly the vibe we're going for.
